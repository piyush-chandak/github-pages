My first static page hosted on github pages

Steps for hosting

  1. Create a repository as public
  2. Add index.html file on root of repository
  3. Go to setting and scroll to Github Pages
  4. Change source from the branch you want to host and select root
  5. Click on save button
  6. Horrayy!!! Your page is hosted on github pages. The url will be https://username/repository_name
